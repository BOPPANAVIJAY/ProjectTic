# ProjectTic
A game in which two players seek in alternate turns to complete a row, a column, or a diagonal with either three O's or three X's drawn in the spaces of a grid of nine squares i.e noughts and crosses.

Rules to play:
1)Have the first player go first. ...
2)Have the second player go second. ...
3)Keep alternating moves until one of the players has drawn a row of three symbols or until no one can win. ...
4)Keep practicing.

How to run the game:
Download the three files provided and paste in any editor(prefer notepad++) that acccess html,css and javascript files.Then run the html file to play the game.
Have Fun !!
